## Tracker Overview

### Column Definitions

| Column | Description |
|--------|-------------|
| **Company** | Startup name |
| **Website** | Verified URL (all 100 have working websites) |
| **Category** | Industry/sector (Fintech, EdTech, HealthTech, etc.) |
| **OSR Area** | Alignment with OSR's 5 key research areas: Education, Insurance, Urban, Health, Culture, or Other |
| **Model** | Business model: B2C, B2B, B2B2C, D2C, Marketplace |
| **Founded** | Year founded (if known) |
| **Funding Stage** | Bootstrapped, Seed, Series A/B/C, Acquired, IPO-bound |
| **Total Raised** | USD funding amount (if disclosed) |
| **Description** | What the startup does (1 line) |
| **Blue Ocean Signal** | Non-customer target, key innovation, or differentiation |
| **Founders** | Founder name(s) |
| **Founder Media Presence** | High / Medium-High / Medium / Low / Unknown |
| **Media Notes** | Active channels, storytelling quality, notable appearances |

### Color Coding

**OSR Research Area:**
- Blue = Education
- Green = Insurance
- Orange = Urban
- Yellow = Health
- Purple = Culture

**Founder Media Presence:**
- Green = High (active storytelling, multiple platforms)
- Yellow = Medium / Medium-High
- Red = Low
- No fill = Unknown

## How to Use This Tracker

### For Deal Sourcing
1. Filter by **OSR Area** to focus on key research domains
2. Sort by **Funding Stage** = "Seed" or "Bootstrapped" for early-stage
3. Check **Blue Ocean Signal** for differentiation potential

### For Content Partnerships
1. Filter **Founder Media Presence** = "High" or "Medium-High"
2. These founders are good candidates for:
   - Podcast interviews
   - Case study collaborations
   - Thought leadership content

### For Research Gaps
1. Filter **Founder Media Presence** = "Unknown"
2. These 74 startups have products but low founder visibility
3. Opportunity: Help them tell their story in exchange for access/insights

### For Blue Ocean Analysis
1. Review **Blue Ocean Signal** column
2. Cross-reference with `09.12.25-blue-ocean-classics-analysis.md`
3. Apply Four Actions Framework: What do they Eliminate/Reduce/Raise/Create?
