# 3D Print

In 3D

## Purpose

Create physical prototypes using 3D printing technology to test form, fit, and function with real users.

**âœ… Ideal for:** Physical product testing without expensive manufacturing tooling

## Preparation

1. **Create 3D design** (CAD software or hire designer)
2. **Choose materials** appropriate for your test (PLA, ABS, resin, etc.)
3. **Select printing method:**
   - DIY (if you have a printer)
   - Online service (Shapeways, Sculpteo)
   - Local maker space
4. **Plan testing sessions** with users

## Execution

1. **Print prototypes** (may need multiple iterations)
2. **Test with users:**
   - Let them handle it
   - Observe usage
   - Collect feedback
3. **Iterate on design** based on findings

## Analysis

- Does the form factor work?
- Is it comfortable to use?
- What would users change?
- Is the size/weight appropriate?
- What materials would the final product need?

## Material Options

| Material | Best For | Cost | Durability |
|----------|----------|------|------------|
| PLA | Quick prototypes | Low | Low |
| ABS | Functional parts | Low | Medium |
| PETG | Durable prototypes | Medium | High |
| Nylon | Moving parts | Medium | High |
| Resin | Fine details | High | Medium |
| Metal | Production-like | Very High | Very High |

## Testing Checklist

### Ergonomics
- [ ] Comfortable to hold?
- [ ] Right size for intended use?
- [ ] Weight feels appropriate?
- [ ] Buttons/controls accessible?

### Functionality
- [ ] Does it do what it's supposed to?
- [ ] Parts fit together?
- [ ] Moving parts work?
- [ ] Durable enough for testing?

### User Reaction
- [ ] Intuitive to use?
- [ ] Aesthetically acceptable?
- [ ] Would they want this product?
- [ ] What would they change?

## Best Practices

### âœ… Do:
- Start with rough prototypes
- Test early and often
- Iterate based on feedback
- Use appropriate materials
- Document each version

### âŒ Don't:
- Over-engineer first prototype
- Skip user testing
- Use expensive materials too early
- Ignore printing constraints
- Expect production quality
