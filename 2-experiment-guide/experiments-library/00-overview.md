# OS Research Experiment Guide

This document contains 33 different experiments to help you validate business ideas and design experiments.

## How to Use This Guide

Each experiment is presented with a consistent structure:
1. **Name & Purpose**
2. **Preparation**
3. **Execution**
4. **Analysis**
5. **Cost**
6. **Set-up Time & Running Time**
7. **Evidence Strength**
8. **Skills Required**
9. **Requirements**

## Evidence Strength Scale

| Rating | Level | Description |
|--------|-------|-------------|
| 1/5 | Very Weak | Opinions, stated preferences |
| 2/5 | Weak | Interest indicators, engagement |
| 3/5 | Medium | Clicks, sign-ups, time spent |
| 4/5 | Strong | Real commitments, deposits |
| 5/5 | Very Strong | Actual purchases, revenue |

## Tips for Best Use

1. Keep this guide open while working on your experiments
2. Use the structure to document your own test results
3. Come back regularly to compare across methods
4. Start with cheaper, faster experiments before investing in expensive ones
5. Combine multiple experiments for stronger evidence
